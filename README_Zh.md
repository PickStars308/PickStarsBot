# PickStarsBot - 摘星辰Bot

Chinese | [English](README.md)

## 描述
欢迎使用PickStarsBot！这是一款基于NoneBot2构建的智能QQ机器人，提供丰富的功能，包括一言、历史上的今天、60秒早报等，快来试试吧！

## 插件
- [x] 一言
- [ ] 历史上的今天
- [x] 60秒早报
- [x] 机器人菜单

## 如何启动

1. 克隆此存储库
```bash
git clone https://github.com/PickStars308/PickStarsBot.git
```
```bash
cd PickStarsBot
```

2. 安装依赖项
```bash
pip install -r requirements.txt
```

3. 启动
```bash
nb run --reload
```
