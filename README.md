## How to start

1. Clone this repository
```bash
git clone https://github.com/PickStars308/PickStarsBot.git
```
```bash
cd PickStarsBot
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Start
```bash
nb run --reload
```
